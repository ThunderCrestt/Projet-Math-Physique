Bonjour voici quelques explications sur notre programme.
Tout d'abord celui ci utilise opengl 3.3 avec glad ainsi que la bibliothèque GLFW, il pourrait donc manquer des affichages sans ces composants.
Le programme devrait alors compiler en ouvant la solution visual studio.
En éxécutant le code une fenêtre s'ouvrira en lançant directement notre blob depuis les airs.
Vous pouvez controller une particule centralle de celui-ci en appuyant sur les touche droite ou gauche des flêches du clavier.
Un escalier est présent pour montrer la chute des divers particules.
